# Social Media App

A simple social media application built with React and Vite. Users can create posts, view a feed, and interact with posts by liking them.

## Features

- Create new posts with text and images
- View a feed of posts
- Like posts
- Styling with Tailwind CSS

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- Lucide React (for icons)
